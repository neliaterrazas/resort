# _Isla Paradisia_

#### _Mock website to display ability to branch and merge, 1/8/19_

#### By _**Travis Toal, Nelia Terrazas**_

## Description

_'Resort' is a project to practice branching repositories with similar content but different styles. It is a website advertising a fake island resort, with interconnected pages. The page does nothing._

## Setup/Installation Requirements

* _View the repository at https://github.com/travisty12/resort_
* _Clone the repository to your desktop_
* _View the code with your preferred text editor, or open with your preferred web browser_

## Known Bugs

_Pictures do not center, even when specified in the style file, for some reason_

## Support and contact details

_If you run into problems with your Isla Paradisia page, it is because it is not a real page_

## Technologies Used

_Nothing but basic HTML and CSS, and the internet for help_
